Introduction

What BentoPump is and what the MAG7 box represents.

BentoPump is a pump.fun protocol for on-chain stock index boxes: ERC-20 box tokens backed by underlying tokenized equities held on-chain. At mint, the intended accounting scale is 1 BOX = 1 USD of underlying tokenized equities, while NAV floats afterward with the live value of the basket.
